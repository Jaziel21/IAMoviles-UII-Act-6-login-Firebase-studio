
# Blueprint

## Overview

This project is a Flutter application that provides a user authentication flow with a welcome screen, a sign-up screen, and a login screen.

## Style and Design

*   **Theme:** Modern and clean design.
*   **Color Palette:**
    *   Primary: Blue
    *   Secondary: White
    *   Text: Black, Grey
*   **Typography:** Clear and legible fonts.
*   **Layout:** Centered and responsive layouts.

## Features

### Welcome Screen

*   Displays a welcome message.
*   Provides options to navigate to the Login or Sign-up screen.
*   Includes a placeholder image.

### Login Screen

*   Allows users to log in with their email and password.
*   Includes a link to the Sign-up screen for new users.
*   Includes a placeholder image.

### Sign-up Screen

*   Allows new users to create an account with a username, email, and password.
*   Includes a link to the Login screen for existing users.
